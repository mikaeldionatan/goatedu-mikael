# Como gerar um Personal Access Token no GitHub

Siga estes passos para gerar um token com permissões corretas:

## Passo 1: Acesse as configurações
1. Vá para https://github.com/settings/tokens
2. Clique em **"Generate new token"** (classic)

## Passo 2: Configure o token
- **Note**: `goatedu-mikael`
- **Expiration**: `90 days` (ou永久)

## Passo 3: Selecione os scopes (importante!)
Marque estas opções:
- ✅ `repo` - Full control of private repositories
- ✅ `read:user` - Read user profile data

## Passo 4: Generate
Clique em **"Generate token"** e copie o token (começa com `ghp_`)

## Passo 5: Forneça o token
Após criar, forneça o token aqui para continuarmos.
