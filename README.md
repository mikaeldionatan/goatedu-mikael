# Guia Gemma 4 no iPhone

IA local, sem internet, rodando direto no seu bolso. Aprenda a instalar, configurar e usar o modelo open-source mais poderoso do Google no iPhone.

Feito por [@mikaeldionatan_](https://www.instagram.com/mikaeldionatan_/)

## Como usar

Basta abrir o arquivo `index.html` no seu navegador.

## Deploy

Você pode fazer deploy desse site em qualquer serviço de hospedagem estática:
- GitHub Pages
- Netlify
- Vercel

## Tecnologias

- HTML5
- CSS3 (Design System personalizado com tema escuro e neon cyan)
- JavaScript Vanilla
